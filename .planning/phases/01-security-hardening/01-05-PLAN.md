    Add this handler:
    ```javascript
    ipcMain.handle('restart-backend', async () => {
      try {
        kill()
        // Read the stored encrypted key and inject into backend env
        let apiKey = ''
        if (safeStorage.isEncryptionAvailable() && fs.existsSync(keyPath())) {
          try { apiKey = safeStorage.decryptString(fs.readFileSync(keyPath())) } catch (_) {}
        }
        startBackend(apiKey)
        await waitForBackend()
        return { ok: true }
      } catch (e) {
        return { ok: false, error: e.message }
      }
    })
    ```

    EDIT 3 — Update startBackend() to accept an optional apiKey parameter.

    Change the startBackend function signature from:
    ```javascript
    function startBackend() {
    ```
    To:
    ```javascript
    function startBackend(apiKey) {
    ```

    Change the env in the spawn call (line 60) from:
    ```javascript
    env: { ...process.env, PORT: String(PORT), PYTHONUNBUFFERED: '1' },
    ```
    To:
    ```javascript
    env: { ...process.env, PORT: String(PORT), PYTHONUNBUFFERED: '1', ...(apiKey ? { ANTHROPIC_API_KEY: apiKey } : {}) },
    ```

    The existing call at line 136 (`startBackend()`) passes no argument — that is correct, it will spread an empty object (no ANTHROPIC_API_KEY at startup). Do NOT change that call site.
